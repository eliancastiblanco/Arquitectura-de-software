# Arquitectura-de-software

## Componente metodológico

Este proyecto tiene como objetivo analizar y diseñar la arquitectura de un sistema de software utilizando buenas prácticas y principios SOLID.

## Objetivos

- Desarrollar una arquitectura escalable.
- Aplicar patrones de diseño.
- Documentar cada fase del desarrollo.
## Tecnologías a utilizar

- Node.js
- Express
- MongoDB
